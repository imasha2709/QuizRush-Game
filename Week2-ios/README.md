# iOS Mini Game Hub (Full Tutorial from Week 1 to Week 4)

A multi-game SwiftUI application featuring three distinct mini-games, local data tracking, location features, and engaging animations.

## Included Games
* **Tap Frenzy:** A fast-paced tapping game to test your reflexes.
* **Light It Up:** A puzzle game focused on grid or pattern illumination.
* **Quiz Rush:** A rapid-fire trivia game against the clock.

## Features
* **Multi-Game Navigation:** Seamless switching using SwiftUI `TabView`.
* **Game Statistics:** Performance tracking visualized using **Swift Charts**.
* **Local Data Persistence:** High scores and settings saved via `UserDefaults`.
* **Map & Location:** **MapKit** and **CoreLocation** integration to find nearby players or gaming hubs.
* **Local Notifications:** **UserNotifications** reminders to keep users engaged.
* **Social Sharing:** Native `ShareLink` integration to share high scores.
* **Dark Mode Support:** Fully responsive native Dark Mode integration for comfortable night gaming.
* **Leaderbord:** A dedicated leaderboard tracking high scores across all three games.

## Tech Stack
* **UI/UX:** SwiftUI, SwiftUI Animations
* **Frameworks:** MapKit, CoreLocation, UserNotifications, Swift Charts
* **Storage:** UserDefaults

## Screens
1. **Home:** The main dashboard to launch any of the three games and the Scoreboard.
2. **Statistics:** Visual charts showing your gaming history and scores.
3. **Map:** View your current location and gaming-related points of interest.
4. **Settings:** Toggle notifications, reset stats, and customize app preferences.
